Id

SetoidFunctionApplyApplicativeChainMonad
The identity data constructor. Useful when a function requires a Functor or a Monad object and you only have a regular value, you can wrap that value in an Id to make the compiler happy.
How to import
import { Id } from 'purify-ts/adts/Id'
